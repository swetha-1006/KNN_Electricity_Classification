# ⚡ KNN Electricity Classification

A Machine Learning project that uses the **K-Nearest Neighbors (KNN) Classification algorithm** to classify electricity-related data based on features provided in the dataset.

## 📌 Project Overview

This project demonstrates how the **KNN (K-Nearest Neighbors)** algorithm can be applied to electricity consumption/classification data to identify patterns and predict categories.

The implementation is done using **Python in Jupyter Notebook** and uses the dataset:

* `finds_electricity_small.csv`

The notebook performs:

* Data loading and preprocessing
* Exploratory Data Analysis (EDA)
* Feature selection
* Data splitting (Training & Testing)
* Model training using KNN
* Prediction and evaluation
